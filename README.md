# LLM Benchmark Platform

A local LLM evaluation platform built using FastAPI and Ollama.

## Features

- Local LLM inference using Ollama
- FastAPI REST API
- Structured JSON outputs
- Pydantic validation
- Automatic retry mechanism
- Benchmarking metrics
  - Latency
  - Token count
  - Tokens per second

## Tech Stack

- Python
- FastAPI
- Ollama
- Pydantic

## Models

- Llama 3.2 3B
